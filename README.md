# dexon-hackathon

client 資料夾下為前端
contracts 是合約

# 部署合約

請先將 secret.js.sample 複製一個 secret.js 並填入錢包的 mnemonic

## 正式環境

npm run migrate

## 測試環境

npm run migrate_local

# 前端測試

cd client & npm run start
